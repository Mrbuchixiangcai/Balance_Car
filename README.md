# Balance_Car
- 平衡车的软硬件制作全教程，包括：上位机，微信小程序，硬件，软件算法
 
## 文件结构：

| 文件名 | 说明 |
|--------|------|
| hardware | 平衡车硬件资料（原理图，PCB） |
| bc_upper | 平衡车上位机代码，上位机采用QT开发，功能：参数设置，平衡车方向控制，PID输出量波形 |
